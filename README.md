# ⚡ Smart Energy Consumption Forecast using ANFIS

This project focuses on predicting household energy consumption using an Adaptive Neuro-Fuzzy Inference System (ANFIS). The model combines the learning capability of neural networks with the reasoning of fuzzy logic to capture complex patterns in time-series power usage data.

The dataset is preprocessed by handling missing values, normalizing features, and extracting temporal attributes such as hour, day, and weekend indicators. A sliding window approach is used to convert the time-series data into a supervised learning format.

The ANFIS model is trained on historical energy consumption along with time-based features to accurately forecast future power usage. This approach helps in understanding consumption patterns and can be useful for smart grid systems, energy optimization, and demand forecasting.

##  Features

* Data preprocessing and normalization
* Time-series feature engineering
* Lag-based input generation
* ANFIS-based prediction model

##  Applications

* Smart energy management
* Load forecasting
* Demand optimization
